Essa tag link css é usada para construção de sites com mais de uma página e que eu queira que a formatação seja a mesma entre as páginas. Pode usar mais de um link css, daí o html5 vai somar oq há nas duas e entregar às páginas.

Pode usar também o style em cada página junto com link css(css externo), deixando preferencias locais para algo, como, por exemplo, o sublinhado no "h1"(css interno) e no "ir para página 02"(css inline)
