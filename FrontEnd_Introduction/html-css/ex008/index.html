Negrito / Destaque:

Nessa frase usando a tag <b>(não semântica): Lorem ipsum dolor sit amet, consectetur adipisicing elit. Pariatur, aspernatur! Consequuntur enim sapiente ab quos, non sit laboriosam porro velit numquam eum amet adipisci totam fugit deserunt consectetur soluta maiores?

Nessa frase usando o strong(possuí semântica e destaque): Lorem ipsum dolor sit amet consectetur adipisicing elit. Atque saepe consectetur velit, provident accusamus deleniti quos voluptates cumque quo ratione magnam ullam repellendus sapiente libero aliquam quis! Fugit, molestias optio!

Itálico / Ênfase

Nessa frase usando a tag <i>: ESSA FRASE EM INTALICO (usa o ctrl + shift + p = abb daí digita i para itálico e n para negrito)

Nessa frase usaremos colocaremos em itálico, porém, com semântica ctrl + shift + p + em: Lorem ipsum dolor sit amet consectetur adipisicing elit. Molestiae atque architecto optio sunt fugiat harum soluta incidunt ullam consequatur maxime, tenetur perferendis, totam, autem aspernatur veritatis repellat accusamus assumenda odio?
